SQL HOUSING Data Cleaning

In this project, housing data is cleaned using SQL Query:

  • Date Format is standardized using Convert() function.
  • PropertyAddress null data is replaced by joining other table that have same data using ISNULL() function.
  • PropertyAddress is breaked into Individual column (Address, City, State) using SUBSTRING(), CHARINDEX() and LEN() function.
  • Alternatively, OwnerAddress is breaked out into Individual column (Address, City, State) using PARSENAME() function.
  • Simplify data into understandable data using CASE statement.
  • Remove duplucate using CTE, ROW_NUMBER() function, PARTITION BY and ORDER BY clauses.
  • Lastly, data is further simplified with deleting unused column using Alter Table, Drop Column Command.
